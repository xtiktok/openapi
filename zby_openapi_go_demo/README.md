# zby_openapi_go_demo
golang demo for zby openapi

#克隆代码

git clone https://github.com/xddlly/zby_openapi_go_demo.git

#进入项目目录

cd zby_openapi_go_demo

#vim main.go  修改直播云提供的 devUrl（openAPI地址）,key（密钥）,applicationID（合作方账号ID）

#运行

go run main.go
